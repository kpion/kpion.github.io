Netmon

Firefox / Chrome extension


View the HTTP headers of the current page, or globally, from all tabs. You can use it at any time, also *after* the page has loaded.

This is useful when there are random errors in some cases and you just want to see what happened, list the past requests.

It also constantly displays current count of requests in any given tab.

In zoom

Firefox / Chrome extension


Zoom in and out on images and videos using shift + mouse wheel

This extension works "inline", i.e. images are not displayed in a popup - instead they are enlarged exactly where they are.
